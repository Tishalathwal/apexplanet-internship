# Task 2 - Intermediate HTML, CSS & JavaScript

Built as part of the **ApexPlanet Web Development Internship** (Summer 2026).

## Objective
Enhance HTML and CSS skills, and learn JavaScript for DOM manipulation.

## Features

### 1. Contact Form
- Input fields: Name, Email, Message
- JavaScript form validation (empty field check + email format check)
- Error messages for invalid inputs
- Success message on successful submission

### 2. Responsive Layout
- **Flexbox** used for navigation bar
- **CSS Grid** used for content layout
- **Media queries** for mobile and tablet responsiveness

### 3. Dynamic To-Do List
- Add tasks dynamically using JavaScript
- Delete individual tasks
- Empty task validation
- Press **Enter** key to add tasks quickly

## Tech Stack
HTML | CSS | JavaScript
